## 概要

### 🎯 コアコンセプト

> **「生きた言語を、生きたニュースから学ぶ」**

教科書の例文ではなく、今日起きている出来事を題材に語学を学ぶことで、実践的で記憶に残る学習体験を提供します。

### 主な機能

- 📰 **ニュースリーダー** - 英語ニュースを読みながら語彙を学習
- 🎓 **レベル判定** - CEFRベースの語彙・読解テスト
- 🔥 **ゲーミフィケーション** - ストリーク、バッジ、XPシステム
- 🤖 **AI学習アシスタント** - MCPベースのインテリジェントな学習支援
- 🌐 **多言語対応** - 英語、スペイン語、韓国語、中国語

### クイックスタート

```bash
# リポジトリをクローン
git clone https://github.com/newslingua/newslingua.git
cd newslingua

# 依存関係をインストール
pnpm install

# 開発サーバーを起動
pnpm dev
```

http://localhost:3000 でアプリにアクセスできます。

### エグゼクティブサマリー

**NewsLingua** は、世界中のニュースを通じて「生きた言葉」を体験しながら語学力を向上させる革新的なプラットフォームです。AIを活用したパーソナライズド学習、Deep News Integration、SNSトレンド分析を通じて、教科書では学べない実践的な語学学習を実現します。

### プロジェクト情報

- **プロジェクト名**: NewsLingua（ニュースリングア）
- **リリース日**: 2026年6月（2026年前期末）
- **対象市場**: 日本国内 → アジア太平洋 → グローバル
- **対応言語**: 英語、スペイン語、韓国語、中国語、フランス語、ドイツ語

### ターゲットユーザー

| セグメント | 特徴 | ニーズ |
|-----------|------|--------|
| ビジネスパーソン（25-45歳） | 海外出張・転職希望 | 実践的なビジネス英語 |
| 学生（16-24歳） | 留学・試験対策 | 効率的な語彙増強 |
| ニュースジャンキー | 毎日ニュースを読む | 多言語でニュース読みたい |
| SNS/エンタメ好き（18-35歳） | セレブ・カルチャーに関心 | スラング・トレンド語彙 |

### ビジネスモデル（フリーミアム4層構造）

| プラン | 月額 | 主な機能 |
|--------|------|----------|
| Free | ¥0 | 5記事/日、100単語、基本ニュース |
| Pro | ¥980 | 無制限、AIアシスト、オフライン |
| Ultimate | ¥1,980 | 全Pro機能、18+コンテンツ、スラング辞書 |
| Business | ¥4,980 | 全機能、チーム管理、API連携 |

### 機能要件

#### レベル判定システム

| レベル | 語彙数目安 | ニュース難易度 | 特徴 |
|--------|-----------|---------------|------|
| A1 | ~1,000語 | 見出しのみ | 基礎単語、簡単な文 |
| A2 | ~2,000語 | 簡易版記事 | 日常語彙、短い記事 |
| B1 | ~4,000語 | 一般ニュース | 中級語彙、標準記事 |
| B2 | ~6,000語 | 詳細記事 | 上級語彙、長文記事 |
| C1 | ~10,000語 | 専門記事 | 専門語彙、分析記事 |
| C2 | 10,000語+ | 原文記事 | ネイティブレベル |

#### コンテンツカテゴリ

- **ニュース**: 政治、経済、国際、テクノロジー、科学
- **エンターテインメント**: セレブ・ゴシップ、映画・ドラマ、音楽、ゲーム
- **スポーツ**: サッカー、野球、バスケ、eスポーツ
- **カルチャー**: アニメ・マンガ、K-POP、ファッション

## MCP統合

### 概要

Model Context Protocol (MCP) を**Python**で実装し、AIアシスタント機能を実現します。

### MCPエコシステム

```
┌───────────────────────────────────────────────────────────────┐
│              Python MCP Server (FastAPI)                      │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐            │
│  │ Vocab   │ │ Grammar │ │ Reading │ │ Progress│            │
│  │ Tools   │ │ Tools   │ │ Tools   │ │ Tools   │            │
│  └─────────┘ └─────────┘ └─────────┘ └─────────┘            │
└───────────────────────────────────────────────────────────────┘
```

### MCPツール一覧

| ツール | 説明 |
|--------|------|
| explain_word | 単語の詳細な説明を生成 |
| generate_examples | 例文を生成 |
| analyze_difficulty | 記事の難易度を分析 |
| summarize_article | 記事を要約 |
| extract_vocabulary | 記事から語彙を抽出 |
| recommend_articles | ユーザーに最適な記事を推薦 |

### プロジェクト構成

```
python-ai-service/
├── app/
│   ├── main.py              # FastAPIエントリーポイント
│   ├── api/                  # REST APIエンドポイント
│   ├── mcp/                  # MCPサーバー・ツール
│   ├── services/             # ビジネスロジック
│   └── core/                 # 設定・セキュリティ
├── Dockerfile
└── pyproject.toml
```
