# NewsLingua ドキュメント

**NewsLingua（ニュースリングア）** は、世界中のニュースを読みながら語学力を向上させる革新的なプラットフォームです。

## 🎯 コアコンセプト

> **「生きた言語を、生きたニュースから学ぶ」**

教科書の例文ではなく、今日起きている出来事を題材に語学を学ぶことで、実践的で記憶に残る学習体験を提供します。

## 主な機能

- 📰 **ニュースリーダー** - 英語ニュースを読みながら語彙を学習
- 🎓 **レベル判定** - CEFRベースの語彙・読解テスト
- 🔥 **ゲーミフィケーション** - ストリーク、バッジ、XPシステム
- 🤖 **AI学習アシスタント** - MCPベースのインテリジェントな学習支援
- 🌐 **多言語対応** - 英語、スペイン語、韓国語、中国語

## クイックスタート

```bash
# リポジトリをクローン
git clone https://github.com/newslingua/newslingua.git
cd newslingua

# 依存関係をインストール
pnpm install

# 開発サーバーを起動
pnpm dev
```

http://localhost:3000 でアプリにアクセスできます。
