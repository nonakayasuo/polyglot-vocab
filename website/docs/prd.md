## エグゼクティブサマリー

**NewsLingua（ニュースリングア）** は、世界中のニュースを通じて「生きた言葉」を体験しながら語学力を向上させる革新的なプラットフォームです。AIを活用したパーソナライズド学習、Deep News Integration、SNSトレンド分析、そして多様なコンテンツ（エンタメ、スラング、18+表現まで）を通じて、教科書では学べない実践的な語学学習を実現します。

### 🎯 コアコンセプト

> **「ニュースを読む」から「ニュースで生きた言葉を体験する」へ**

教科書の例文ではなく、今日起きている出来事を題材に、フォーマルな表現からスラング・ミームまで、言語の全スペクトラムを学ぶことで、実践的で記憶に残る学習体験を提供します。

### プロジェクト名

**NewsLingua**（ニュースリングア）
- News（ニュース）+ Lingua（言語、ラテン語）
- 「ニュースを通じて言語を学ぶ」というコンセプトを直接表現
- グローバルに通用する発音しやすい名前

### 商用リリース目標

- **リリース日**: 2026年6月（2026年前期末）
- **対象市場**: 日本国内 → アジア太平洋 → グローバル
- **初期対応言語**: 英語、スペイン語、韓国語、中国語、フランス語、ドイツ語

### 🎯 レベル判定システム（コア機能）

#### 初回レベル診断

```
┌─────────────────────────────────────────────────────────────────┐
│                    レベル判定フロー                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  1. 簡易質問（2分）                                              │
│     • 学習経験  • 目標  • 興味分野  • スラング理解度              │
│                                                                  │
│  2. 語彙テスト（5分）                                            │
│     • 段階的難易度  • A1→C2                                      │
│     • フォーマル語彙 + スラング/口語テスト                        │
│                                                                  │
│  3. 読解テスト（5分）                                            │
│     • 短い記事  • 理解度チェック                                  │
│     • ニュース記事 + SNS投稿理解                                 │
│                                                                  │
│  4. レベル判定結果                                               │
│     • CEFRレベル（A1-C2）                                        │
│     • 強み・弱み分析                                             │
│     • 推奨学習プラン                                             │
│     • スラング理解度スコア                                       │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

#### レベル判定の仕組み

| レベル | 語彙数目安 | ニュース難易度 | スラング対応 | 特徴 |
|--------|-----------|---------------|-------------|------|
| A1 | ~1,000語 | 見出しのみ | なし | 基礎単語、簡単な文 |
| A2 | ~2,000語 | 簡易版記事 | 基本口語 | 日常語彙、短い記事 |
| B1 | ~4,000語 | 一般ニュース | 一般スラング | 中級語彙、標準記事 |
| B2 | ~6,000語 | 詳細記事 | SNS表現 | 上級語彙、長文記事 |
| C1 | ~10,000語 | 専門記事 | ミーム・ネットスラング | 専門語彙、分析記事 |
| C2 | 10,000語+ | 原文記事 | 全レジスター | ネイティブレベル |

#### 継続的レベル調整

- 学習データから自動レベル更新
- 定期的な再診断オプション
- 分野別レベル（ビジネス/科学/スポーツ/エンタメ等）
- **レジスター別レベル（フォーマル/カジュアル/スラング）**

### 技術スタック

| レイヤー | 技術 | 理由 |
|---------|------|------|
| フロントエンド | Next.js 16+ / React 19 | SSR/SSG、高速 |
| モバイル | Flutter + Dart 3 | BtoC向け高パフォーマンス、60fpsスムーズ動作 |
| API | Hono (TypeScript) | 軽量・高速 |
| AI Service | **FastAPI (Python)** | AI/MLエコシステム |
| MCP | **Python MCP SDK** | AIツール統合 |
| 認証 | Better Auth | セルフホスト・柔軟 |
| DB | PostgreSQL (Neon) | サーバーレス |
| ベクトルDB | Pinecone | 意味検索 |
| 決済 | Stripe | グローバル対応 |
| **SNS統合** | Twitter API, Reddit API, TikTok API | トレンド取得 |
| **スラング辞書** | Urban Dictionary API + 自社DB | 口語表現 |

## リスクと対策

### 技術リスク

| リスク | 影響 | 対策 |
|--------|------|------|
| ニュースAPI制限 | 高 | 複数ソース、キャッシュ戦略 |
| AI APIコスト | 高 | Python最適化、ローカルモデル検討 |
| Python-TypeScript連携 | 中 | 明確なAPI契約、テスト充実 |
| **SNS API制限/変更** | 高 | 複数ソース、スクレイピングバックアップ |
| **スラングDB品質** | 中 | コミュニティ投稿 + AI検証 |

### ビジネスリスク

| リスク | 影響 | 対策 |
|--------|------|------|
| ニュース著作権 | 高 | 正式ライセンス取得 |
| 競合模倣 | 中 | ニュース×AI×MCPの独自性 |
| **18+コンテンツ規制** | 高 | 法的確認、年齢確認厳格化 |
| **不適切コンテンツ** | 中 | AIモデレーション、報告機能 |
