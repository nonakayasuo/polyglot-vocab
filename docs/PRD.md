## エグゼクティブサマリー

**NewsLingua（ニュースリングア）** は、世界中のニュースを読みながら語学力を向上させる革新的なプラットフォームです。AIを活用したパーソナライズド学習、MCP統合によるインテリジェントなアシスタント機能、そしてグローバルなコミュニティを通じて、効率的で楽しい語学学習を実現します。

### 🎯 コアコンセプト

> **「生きた言語を、生きたニュースから学ぶ」**

教科書の例文ではなく、今日起きている出来事を題材に語学を学ぶことで、実践的で記憶に残る学習体験を提供します。

### プロジェクト名

**NewsLingua**（ニュースリングア）
- News（ニュース）+ Lingua（言語、ラテン語）
- 「ニュースを通じて言語を学ぶ」というコンセプトを直接表現
- グローバルに通用する発音しやすい名前

### 商用リリース目標

- **リリース日**: 2026年6月（2026年前期末）
- **対象市場**: 日本国内 → アジア太平洋 → グローバル
- **初期対応言語**: 英語、スペイン語、韓国語、中国語、フランス語、ドイツ語

## 機能要件

### 🎯 レベル判定システム（コア機能）

#### 初回レベル診断

```
┌─────────────────────────────────────────────────────────────┐
│                    レベル判定フロー                           │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  1. 簡易質問（2分）                                          │
│     • 学習経験  • 目標  • 興味分野                           │
│                                                              │
│  2. 語彙テスト（5分）                                        │
│     • 段階的難易度  • A1→C2                                  │
│                                                              │
│  3. 読解テスト（5分）                                        │
│     • 短い記事  • 理解度チェック                             │
│                                                              │
│  4. レベル判定結果                                           │
│     • CEFRレベル（A1-C2）                                    │
│     • 強み・弱み分析                                         │
│     • 推奨学習プラン                                         │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

#### レベル判定の仕組み

| レベル | 語彙数目安 | ニュース難易度 | 特徴 |
|--------|-----------|---------------|------|
| A1 | ~1,000語 | 見出しのみ | 基礎単語、簡単な文 |
| A2 | ~2,000語 | 簡易版記事 | 日常語彙、短い記事 |
| B1 | ~4,000語 | 一般ニュース | 中級語彙、標準記事 |
| B2 | ~6,000語 | 詳細記事 | 上級語彙、長文記事 |
| C1 | ~10,000語 | 専門記事 | 専門語彙、分析記事 |
| C2 | 10,000語+ | 原文記事 | ネイティブレベル |

#### 継続的レベル調整

- 学習データから自動レベル更新
- 定期的な再診断オプション
- 分野別レベル（ビジネス/科学/スポーツ等）

### 技術スタック

| レイヤー | 技術 | 理由 |
|---------|------|------|
| フロントエンド | Next.js 16+ / React 19 | SSR/SSG、高速 |
| モバイル | Flutter + Dart 3 | BtoC向け高パフォーマンス、60fpsスムーズ動作 |
| API | Hono (TypeScript) | 軽量・高速 |
| AI Service | **FastAPI (Python)** | AI/MLエコシステム |
| MCP | **Python MCP SDK** | AIツール統合 |
| 認証 | Better Auth | セルフホスト・柔軟 |
| DB | PostgreSQL (Neon) | サーバーレス |
| ベクトルDB | Pinecone | 意味検索 |
| 決済 | Stripe | グローバル対応 |
